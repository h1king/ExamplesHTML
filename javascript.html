Jquery

JQuery is a javaScript library designed to simplify the client-side scripting of HTML. This must be include in header of web page where jquery will be used

Include

<script src="http://code.jquery.com/jquery-latest.js"></script>

Basic selectors

$("*") // Select All elements

$("[input]", ".class") // Multiple selector

For actions like change value of label,show/hide elements, change color and enable disable elements like buttons the following properties help this

    $("[id$=elementID]").addClass("addedClass"); //add css class to element
    $("[id$=elementID]").removeClass("class"); //remove css class to element
    $("[id$=elementID]").hide(); //hide element
    $("[id$=elementID]").show(); //show element
    $("[id$=elementID]").removeAttr("disabled"); //remove attribute from element
    $("[id$=elementID]").attr("disabled","disabled"); //change attribute value to element

Handlebars

Handlebars is a javascript library for template html pages. with handlebars it's possible to pass a json data to template and the page was generated based on this template. A example of handlebars template is composed by html tags and for dynamic elements with bracelets like {{ data element }}.

Reference to Handlebars

<script src="https://github.com/downloads/wycats/handlebars.js/handlebars-1.0.rc.1.js"></script>

Sample template

<script id="person-template" type="text/x-handlebars-template">
<div class="person">
  <h1>{{Name}}</h1>
  <h5>Age:{{Age}}</h5>
  <h5>Location:{{Location}}</h5>
  <h5>Origin:{{Origin}}</h5>
</div>
</script>

Fill this template with data it's necessary to identify the template, data and location of template.

var source = $("#"+templateId+"").html();//get html of template
var template = Handlebars.compile(source);//handlebars compile the template
var html = template(data); // add data to template
$("[id$="+destinationId+"]").empty(); //clean template area
$("[id$="+destinationId+"]").html(html); //Add html generated with temaplate and data to final location

Exist some helper functions that could be created to make some data changes and help with data representation

Handlebars.registerHelper('link', function(text, url) {
  text = Handlebars.Utils.escapeExpression(text);
  url  = Handlebars.Utils.escapeExpression(url);

  var result = '<a href="' + url + '">' + text + '</a>';

  return new Handlebars.SafeString(result);//if the result is html the return should have SafeString.
});

List

ListJS it's a javascript library that help us to make any list searchable, sortable and filterable. Also we can define the number of items that can be showed in the "table"

Reference to ListJS

<script type="text/javascript" src="js/list.js"></script>

Reference to the plugin to do the paging

<script type="text/javascript" src="js/list.paging.min.js"></script>
